التفوق — نسخة جاهزة للنشر على GitHub Pages مع Firebase (MVP)
ملفات المشروع داخل هذا الأرشيف:

- index.html
- styles.css
- firebase-config.js  (معبأ من إعدادات مشروعك)
- app-full.js

كيف تنشر:
1. أنشئ repo جديد على GitHub (public).
2. ارفع محتويات هذا المجلد إلى الفرع main (ارفع الملفات مباشرة عبر واجهة GitHub).
3. في GitHub → Settings → Pages → اختر الفرع main و/أو root ثم احفظ.
4. افتح الموقع الناتج وجرب التسجيل والدخول.

ملاحظات أمان:
- حالياً Firestore قواعدها في وضع الاختبار — بعد الاختبار غيّر Rules.
- firebase-config.js يحتوي مفاتيح مشروعك (عادة آمنة للويب)، لا تشاركها علناً لو حساس.
